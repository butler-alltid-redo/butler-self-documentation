---
title: How this site is built
---

This site is built with **Jupyter Book 2** (the MyST ecosystem).

## Source layout

- `myst.yml` — project + site config and table of contents
- `*.md` — content pages (MyST Markdown)

## Build

Locally:

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
jupyter-book build
```

The HTML output ends up under `_build/html/`.

## Deploy

Deployment is handled via **GitHub Pages** using a GitHub Action generated by:

```bash
jupyter-book init --gh-pages
```

The intent is: keep source on `main`, publish the built site on a separate deployment target (GitHub Pages), without committing build artifacts to `main`.
